Wall types in depth

Wall type

Reinforced Earth Wall (RE)

Henri Vidal 1963 system: steel-strip reinforcement, granular sand backfill, cruciform precast facing. The original MSE category.

Read guide → Wall type

Reinforced Soil Wall (RS)

Modern variants: HDPE geogrid, PET woven grid, welded steel wire mesh, polymeric strip. The full MSE family.

Read guide → Wall type

Shored MSE Hybrid (SMSE)

Soil-nail or anchored shoring combined with MSE facing for tight cuts where standard MSE cannot reach. FHWA NHI-09-087.

Read guide → Wall type

Counterfort RC Wall

Tall RC walls with vertical stiffening ribs. Stem flexure dropped via horizontal span between counterforts.

Read guide → Wall type

Gravity Retaining Wall

Mass concrete and masonry. The classical short-wall and heritage solution.

Read guide → Wall type

Crib Wall

Interlocking precast concrete gravity wall. Stretchers, headers, granular fill. Landscape and amenity applications.

Read guide → Wall type

Gabion Wall

Wire-basket stone retaining wall. Riverbank, slope toe, remote sites. Permeable, flexible, fast to build.
